Core Concepts

  • All matter is made up of atoms.
  • Atoms are the smallest amount of an element.
  • An element is made up of atoms that are all the same type.
  • Atoms are made up protons, neutrons and electrons.
  • The nucleus of an atom is made up of its protons and neutrons.
  • Electrons move around the nucleus.
  • The number of protons in the nucleus of the atom determines what type of atom it is.
  • Typically, an atoms has the same number of electrons that it has protons.
  • An atom can gain or lose electrons and when it does, it is called an ion.
  • An atom cannot gain or lose protons without becoming a new type of atom, because the number of protons determines the type of atom it is.
  • The periodic table lists all of the elements by atomic number, in columns that reflect the chemical properties of the elements.
  • The chemical properties of an element are controlled by the number of electrons in its outer shell.

Terms to Know

See if you know these terms, then click on the buttons to reveal the definitions.


An atom is the smallest piece of an element.



An element is made up of only one type of atom.



A proton is a positively charged particle in the nucleus of an atom. The number of protons in the nucleus determines what type of atom it is.



A neutron is a neutral particle (it has no charge) in the nucleus of an atom. The neutrons help hold the protons in the nucleus together.



An electron is a negatively charged particle that moves around the nucleus of an atom. Typically, an atom has the same number of electrons that it has protons so the atom has no overall charge.



The nucleus is the center of the atom and contains its protons and neutrons.



An ion is an atom that has gained or lost electrons, so it has more or fewer electrons than it does protons. The number of protons in the nucleus still determines what type of ion it is.



The electrons move around the nucleus of an atom in what are thought of as "electron shells". Each shell is limited in how many electrons it can hold. The first shell can only hold 2 electrons, the second shell can hold 8.
