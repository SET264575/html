Core Concepts

  • All matter is made up of atoms.
  • Atoms are the smallest amount of an element.
  • An element is made up of atoms that are all the same type.
  • Atoms are made up protons, neutrons and electrons.
  • The nucleus of an atom is made up of its protons and neutrons.
  • Electrons move around the nucleus.
  • The number of protons in the nucleus of the atom determines what type of atom it is.
  • Typically, an atoms has the same number of electrons that it has protons.
  • An atom can gain or lose electrons and when it does, it is called an ion.
  • An atom cannot gain or lose protons without becoming a new type of atom, because the number of protons determines the type of atom it is.
  • The periodic table lists all of the elements by atomic number, in columns that reflect the chemical properties of the elements.
  • The chemical properties of an element are controlled by the number of electrons in its outer shell.

Terms to Know

See if you know these terms, then click on the buttons to reveal the definitions.


An atom is the smallest piece of an element.



An element is made up of only one type of atom.



A proton is a positively charged particle in the nucleus of an atom. The number of protons in the nucleus determines what type of atom it is.



A neutron is a neutral particle (it has no charge) in the nucleus of an atom. The neutrons help hold the protons in the nucleus together.



An electron is a negatively charged particle that moves around the nucleus of an atom. Typically, an atom has the same number of electrons that it has protons so the atom has no overall charge.



The nucleus is the center of the atom and contains its protons and neutrons.



An ion is an atom that has gained or lost electrons, so it has more or fewer electrons than it does protons. The number of protons in the nucleus still determines what type of ion it is.



The electrons move around the nucleus of an atom in what are thought of as "electron shells". Each shell is limited in how many electrons it can hold. The first shell can only hold 2 electrons, the second shell can hold 8.


Other Concepts We Didn't Talk About

  • isotopes
    •         The number of protons in an atom determines what element it is.  If there are more or fewer protons, then it is a different element.  But the number
        of neutrons in the nucleus can be different for different atoms of the same element.  For instance, carbon can have 6, 7 or 8 neutrons in its 
        nucleus.  These different types of carbon are called different isotopes of carbon.  All elements have different isotopes.  The different isotopes 
        of an element all have the same chemical properties because those properties are determined by the number of electrons it has.  The number of
        neutrons in the nucleus affect the weight of the atom and also affect the stability of the nucleus.
  • radioactive decay
    •         Some isotopes are unstable, which means their nuclei will break down, which is called "radioactive decay."  Sometimes when a nucleus decays, it 
        gains or loses a proton and becames a different element.  Radioactive decay is why some isotopes are "radioactive".  The study of radioactive 
        decay forms the basis of nuclear physics and is not usually considered part of chemistry.
  • quantum mechanics
    •         The model of an atom we have been using is called the "Bohr model".  It is very useful, but it is also not correct.  The structure of an atom is 
        described by a scientific theory called "quantum mechanics".  Quantum mechanics explains the movement of electrons around the nucleus (the
        electrons don't really move in nice orbits.)  It also explains the number of electrons in each electron shell.

The Atomic Nucleus


    The nucleus of an atom is made of protons and neutrons.  The number of protons determines what type of atom it is.  This is an illustration of a
    carbon atom which has 6 protons.  The number of electrons is the same as the number of protons to balance out the charge.

Reading the Periodic Table


    Each cell in the periodic table shows what the element's chemical symbol is, what the name of the element is, and how many protons it has in its
    nucleus.  Some periodic tables also contain a lot of other information about the element.

The Periodic Table


    Many periodic tables are colored to show the properties of the elements.  Some elements are metals, some are non-metals, some are gases, etc.

The Lanthanide and Actinide Series


    Most periodic tables are arranged with the lanthanide and actinide series set off separately below the main table.  This is can seem confusing
    and make it seem like these elements don't fit in the periodic table.  Actually, this is only done for convenience because when they are put in their 
    proper place, the periodic table is very wide.

Valence


    The number of electrons in the outer shell, which is called its valence, determines the chemical properties of the element.
